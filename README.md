# Iris Data Analysis using Excel

Project Overview

This project analyzes the Iris dataset using Microsoft Excel. Pivot Tables were used to summarize the dataset and understand the distribution of different iris species.

Tools Used

- Microsoft Excel
- Pivot Tables
- Basic Statistical Analysis

Project Work

- Created Pivot Tables to analyze species distribution
- Calculated average measurements of iris features
- Organized the dataset for easier analysis
- Created charts for data visualization

Dataset

The Iris dataset contains measurements of iris flowers including:

- Sepal Length
- Sepal Width
- Petal Length
- Petal Width
- Species

Outcome

This project demonstrates basic data analysis using Excel Pivot Tables and charts to summarize and visualize the dataset.
